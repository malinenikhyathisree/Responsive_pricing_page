# Responsive Pricing Table UI

This is a simple and responsive Pricing Table webpage created using HTML and CSS Flexbox.

## Features
- Responsive Flexbox Layout
- Clean UI Design
- Three Pricing Plans
- Mobile Responsive Design
- Beginner Friendly Frontend Project

## Technologies Used
- HTML5
- CSS3
- Flexbox

## Project Preview
The webpage contains:
- Basic Plan
- Standard Plan
- Premium Plan

Each pricing card includes:
- Storage details
- User limits
- Support features
- Sign Up button

## Learning Outcome
Through this project, I practiced:
- Flexbox layout
- Responsive web design
- CSS styling
- Media queries
- UI structuring
